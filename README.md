# 👟 Rolda Toro - E-Commerce de Calzado

**Rolda Toro** es un proyecto universitario de desarrollo web que simula una tienda en línea completa (E-commerce) dedicada a la venta de calzado. El proyecto está enfocado en brindar una experiencia de usuario (UX) excepcional y una interfaz de usuario (UI) moderna, visualmente atractiva y fácil de navegar.

---

## 📖 Descripción del Proyecto

Este proyecto nace como una iniciativa académica para aplicar conceptos de desarrollo web frontend (y/o backend) en un entorno de negocio realista. La plataforma permite a los usuarios explorar un catálogo diverso de zapatos divididos por categorías, visualizar detalles de cada producto y gestionar sus compras a través de un carrito interactivo.

La filosofía de la marca ficticia "Rolda Toro" se centra en la **comodidad, el estilo, materiales sostenibles y detalles artesanales**.

## ✨ Características Principales (Features)

- **🛒 Carrito de Compras Dinámico:** Los usuarios pueden agregar, visualizar y gestionar los productos que desean comprar antes de finalizar la transacción. El contador se actualiza en tiempo real en la barra de navegación.
- **📂 Categorización Intuitiva:** Navegación fluida segmentada por colecciones: *Hombre, Mujer, Niños* y sección de *Ofertas*.
- **🎨 Diseño UI/UX Atractivo:** Interfaz moderna y amigable con el usuario. Uso de una paleta de colores cálida (terracota, beige y blanco) que transmite elegancia y comodidad.
- **📱 Totalmente Responsivo:** Diseño adaptable a diferentes tamaños de pantalla (Móviles, Tablets y Escritorio). *(Añade esto si tu página es responsive)*.
- **🔍 Catálogo Detallado:** Tarjetas de productos que muestran imagen, nombre, precio, descripción corta y botones de acción rápida ("Agregar" y "Ver").
